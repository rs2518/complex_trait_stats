0_vi_downloads.sh
=================
- Download variant information files from GeneAtlas for all non-sex chromosomes (files have prefix 'snps.imputed....')
- One-time download


1_trait_download.sh
===================
- Download summary statistics for up to 10 given trait keys at a time (See 'Traits Table  GeneATLAS.csv' for details)
- Returns 'results/' directory with imputed results files for all chromosomes excluding sex chromosomes


Traits_Table_GeneATLAS.csv
===========================
- Spreadsheet of trait keys and their respective description
- Original file name: 'Traits Table  GeneATLAS.csv'. File renamed to remove special characters
